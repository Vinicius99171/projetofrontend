O site funciona como um caderno de contas digital. Ele começa quando você registra qualquer tipo de transação, você preenche os dados pedidos e depois clica em "adicionar, o site
vai pegar essa informação e guardar como um registro, o trabalho do código é calcular de forma prática e eficiente, pegando todos os registros salvos e somando o que é a entrada e
subtraindo a saída. Quando passar por isso o resultado vai ser mostrado no "Saldo Atual". Em resumo o site vai salvar a lista no seu PC, recalculando sempre o total e mostrando o
seu saldo atualizado.
A lógica dele é garantir que os dados não sumam quando o usuário fechar a página, mesmo sem ter um servidor ou um banco de dados, ou seja quando você fecha o seu navegador, ainda
sim os seus dados vão ser mantidos se eles tiverem sido salvos.
